# JobRoadMap (JRM)

데이터 기반 맞춤 IT 학습 가이드 웹서비스의 프로토타입입니다. 채용 공고(JD)를 입력하면 선수지식 → 핵심기술 → 실무스택 → 보너스스킬 구조로 재정렬된 로드맵을 생성하고, 하루 학습 시간과 기간을 반영해 주차별 학습 플랜을 제공합니다. 또한 매일 0시에 실행되는 간단한 크롤러가 샘플 공고를 파싱해 기술 키워드 데이터를 준비합니다.

## 구성
- **backend/**: Java 17 + Spring Boot 3.3 기반 REST API
  - `/api/roadmap/from-jd` : JD 텍스트 기반 로드맵 생성
  - `/api/crawl/daily` : 샘플 HTML을 크롤링해 키워드/공고 목록 제공
- **frontend/**: HTML/CSS/Vanilla JS 기반 단일 페이지
  - 백엔드 API를 호출해 로드맵/크롤링 데이터를 시각화

## 실행 방법
1. **백엔드 실행**
   - Java 17과 Maven이 설치되어 있어야 합니다.
   - 터미널에서 백엔드 디렉터리로 이동 후 실행:
     ```bash
     cd backend
     mvn spring-boot:run
     ```
   - 서버는 기본적으로 `http://localhost:8080`에서 실행됩니다.

2. **프론트엔드 실행**
   - 별도 빌드 없이 정적 파일로 동작합니다.
   - 새 터미널에서 프로젝트 루트(frontend 상위)에서 간단히 개발용 서버를 띄웁니다.
     ```bash
     cd frontend
     python -m http.server 8000
     ```
   - 브라우저에서 `http://localhost:8000`에 접속합니다.

3. **사용 흐름**
   - 프론트에서 JD 텍스트, 준비 기간, 하루 학습 시간을 입력하고 **로드맵 생성하기**를 눌러 `/api/roadmap/from-jd`를 호출합니다.
   - **크롤링 데이터 보기** 버튼으로 `/api/crawl/daily` 결과(샘플 공고/키워드)를 확인할 수 있습니다.

## 참고
- 크롤링은 `sample-posts.html`을 파싱하는 형태로 구현되어 있어 네트워크 없이도 동작합니다. 실제 서비스로 확장 시 Jsoup 기반으로 실서비스 사이트를 파싱하도록 교체하면 됩니다.
- GPT 연동, 인증/DB 등은 확장 포인트로 남겨 두었습니다.
